# Portfolio Repository

Dit portfolio dient als centrale plek waar ik mijn werk, vaardigheden en groei presenteer. De inhoud is ontwikkeld met behulp van Visual Studio Code en bestaat uit meerdere pagina's waarop verschillende aspecten van mijn profiel worden toegelicht.

## Inhoud

### **Home**

Een introductie van wie ik ben. Deze pagina biedt bezoekers een eerste indruk van mijn achtergrond en doelstellingen.

### **Over mij**

Een uitgebreidere beschrijving van mijn opleiding, interesses en vaardigheden. Hier wordt nader toegelicht welke richting ik binnen softwareontwikkeling ambieer en welke competenties ik ontwikkeld heb.

### **Projecten**

Een overzicht van eerder gerealiseerde (groeps-)projecten. Per project worden de volgende onderdelen opgenomen:

* Een beschrijving van de functionaliteit en het doel
* Screenshots ter illustratie van de werking
* Links naar de bijbehorende broncode

### **Contact**

Verwijzingen naar mijn e-mailadres en GitHub-profiel.
